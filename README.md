EdTech & Programming Services Landing Page

A modern, responsive landing page designed for an Educational Technology platform. This project showcases advanced UI/UX principles, glassmorphism aesthetics, and a mobile-first responsive architecture.

🚀 Live Demo
https://ishaambasta.github.io/EdTech/

✨ Key Features

Responsive Hero Section: High-impact entrance with a parallax background and a vibrant gradient overlay for optimal text readability and visual energy.

Dynamic Services Grid: Interactive service cards featuring CSS-driven hover animations, custom iconography, and a primary-color accent bar for a professional look.

Fixed Testimonial Carousel: A fully functional, touch-responsive student feedback slider powered by Bootstrap JS, ensuring smooth transitions between reviews.

Integrated Contact System: A modern two-column contact section including a localized information panel, a functional inquiry form, and an embedded Google Maps integration.

Glassmorphism UI: Modern header and UI elements utilizing backdrop filters and alpha-channel transparency for a premium, layered feel.

🛠️ Technology Stack

Frontend Framework: Bootstrap 5.3 (Grid system, Carousel component, Utility classes)

Styling: Custom CSS3 with CSS Variables (Design Tokens) and Glassmorphism techniques.

Typography: Inter via Google Fonts for a clean, professional aesthetic.

Iconography: Font Awesome 6.x for scalable vector icons.

Interactivity: Vanilla JavaScript & Bootstrap JS Bundle for component logic and animations.

🧠 Technical Highlights

Design Tokens: Utilized CSS variables for consistent branding across the primary pink palette, making theme scalability effortless.

Visual Depth: Implemented layering using z-index and subtle radial gradients to create a sense of hierarchy and focus.

Mobile-First Design: Leveraged Bootstrap's responsive breakpoints to ensure the interface adapts seamlessly from desktops to mobile devices.

Component Connectivity: Properly synchronized data attributes and element IDs to ensure JavaScript-dependent components (like the Carousel) function reliably.
